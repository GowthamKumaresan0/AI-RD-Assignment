# AI R&D Internship Assignment

## Overview

This repository includes my solution for the AI Research & Development Internship Assignment.
The goal of this project is to determine the unknown parameters of a parametric curve from a collection of unordered two-dimensional points.The implementation is focused on reconstructing the curve and using numerical optimization methods to find the best-fit parameters for the given dataset.

---

## Problem Statement

The parametric curve is defined by the following equations:

$$
x(t)=t\cos(\theta)-e^{M|t|}\sin(0.3t)\sin(\theta)+X
$$

$$
y(t)=42+t\sin(\theta)+e^{M|t|}\sin(0.3t)\cos(\theta)
$$

where the unknown parameters satisfy the following constraints:

| Parameter | Range |
|-----------|-------|
| $t$ | $6 \le t \le 60$ |
| $\theta$ | $0^\circ < \theta < 50^\circ$ |
| $M$ | $-0.05 < M < 0.05$ |
| $X$ | $0 < X < 100$ |

The objective is to estimate the values of **Оё**, **M**, and **X** that minimize the reconstruction error between the predicted curve and the provided dataset.

## Theoretical Approach

The problem can be interpreted as a nonlinear parameter estimation problem. A general theoretical solution consists of the following stages:

### 1. Curve Reconstruction

The provided dataset contains unordered points sampled from an unknown traversal of the curve. Before estimating the unknown parameters, a continuous ordering of the points must be reconstructed. This establishes a meaningful representation of the underlying geometric structure.

### 2. Parameterization

Once the curve has been reconstructed, each point is associated with a parameter value \(t\). A common strategy is arc-length parameterization, where the parameter increases proportionally with the accumulated distance along the reconstructed curve.

### 3. Mathematical Modeling

The ordered points are represented using the given parametric equations while treating **Оё**, **M**, and **X** as unknown variables.

### 4. Parameter Estimation

An objective function is formulated to measure the difference between the observed points and the points generated by the mathematical model. Since the equations are nonlinear with respect to the unknown parameters, analytical solutions are generally impractical. Consequently, numerical optimization methods are employed to estimate the optimal parameters.

### 5. Validation

The estimated parameters are validated by comparing the reconstructed curve against the original dataset using an appropriate error metric. A smaller reconstruction error indicates that the estimated parameters more accurately describe the underlying curve.

## Methodology

The solution follows the workflow below:

1. Load the unordered dataset.
2. Reconstruct a continuous ordering of the points using a nearest-neighbor traversal.
3. Implement the parametric curve model.
4. Define an objective function based on the reconstruction error.
5. Estimate the unknown parameters using the Differential Evolution optimization algorithm available in SciPy.
6. Report the optimized parameters and reconstruction loss.

## Optimization Strategy

The optimization searches for the following unknown parameters:

- Оё (Theta)
- M
- X

The search is performed within the following bounds:

| Parameter | Lower Bound | Upper Bound |
|-----------|------------:|------------:|
| Оё | 0В° | 50В° |
| M | -0.05 | 0.05 |
| X | 0 | 100 |

The objective function minimizes the average Euclidean distance between the observed points and the points generated by the parametric model.
