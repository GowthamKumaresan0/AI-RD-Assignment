# AI R&D Internship Assignment

## Overview

This repository contains my solution for the **AI Research & Development Internship Assignment**. The objective of this project is to estimate the unknown parameters of a parametric curve from a set of unordered two-dimensional points. The implementation focuses on reconstructing the curve and applying numerical optimization techniques to recover the unknown parameters that best fit the given dataset.

---

## Problem Statement

The parametric curve is defined by the following equations:

$$
x(t)=t\cos(\theta)-e^{M|t|}\sin(0.3t)\sin(\theta)+X
$$

$$
y(t)=42+t\sin(\theta)+e^{M|t|}\sin(0.3t)\cos(\theta)
$$

where the unknown parameters satisfy the following constraints:

| Parameter | Range |
|-----------|-------|
| $t$ | $6 \le t \le 60$ |
| $\theta$ | $0^\circ < \theta < 50^\circ$ |
| $M$ | $-0.05 < M < 0.05$ |
| $X$ | $0 < X < 100$ |

The objective is to estimate the values of **θ**, **M**, and **X** that minimize the reconstruction error between the predicted curve and the provided dataset.

## Methodology

The solution follows the workflow below:

1. Load the unordered dataset.
2. Reconstruct a continuous ordering of the points using a nearest-neighbor traversal.
3. Implement the parametric curve model.
4. Define an objective function based on the reconstruction error.
5. Estimate the unknown parameters using the Differential Evolution optimization algorithm available in SciPy.
6. Report the optimized parameters and reconstruction loss.

---

## Optimization Strategy

The optimization searches for the following unknown parameters:

- θ (Theta)
- M
- X

The search is performed within the following bounds:

| Parameter | Lower Bound | Upper Bound |
|-----------|------------:|------------:|
| θ | 0° | 50° |
| M | -0.05 | 0.05 |
| X | 0 | 100 |

The objective function minimizes the average Euclidean distance between the observed points and the points generated by the parametric model.
