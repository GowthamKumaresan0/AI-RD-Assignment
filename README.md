# AI R&D Internship Assignment

## Overview

This repository contains my solution for the **AI Research & Development Internship Assignment**. The objective is to estimate the unknown parameters of a parametric curve using a dataset of unordered two-dimensional points. The implementation combines curve reconstruction and numerical optimization to recover the parameters that best describe the given data.

---

## Problem Statement

The given parametric curve is defined by

$$
x(t)=t\cos(\theta)-e^{M|t|}\sin(0.3t)\sin(\theta)+X
$$

$$
y(t)=42+t\sin(\theta)+e^{M|t|}\sin(0.3t)\cos(\theta)
$$

where the unknown parameters satisfy the following constraints:

| Parameter | Range |
|-----------|-------|
| $t$ | $6 \le t \le 60$ |
| $\theta$ | $0^\circ < \theta < 50^\circ$ |
| $M$ | $-0.05 < M < 0.05$ |
| $X$ | $0 < X < 100$ |

The objective is to determine the values of **θ**, **M**, and **X** that best reconstruct the given dataset.

## Methodology

The implemented workflow consists of the following stages:

1. Load the dataset containing unordered `(x, y)` coordinates.
2. Reconstruct a continuous traversal of the point cloud using a nearest-neighbor ordering algorithm.
3. Implement the mathematical model of the parametric curve.
4. Define an optimization objective based on the reconstruction error between the predicted and observed points.
5. Estimate the unknown parameters using the Differential Evolution algorithm available in SciPy (Virtanen et al. 2020).

---

## Optimization Strategy

The optimization searches for the following unknown variables:

- θ (Theta)
- M
- X

Parameter bounds:

| Parameter | Lower Bound | Upper Bound |
|-----------|------------:|------------:|
| θ | 0° | 50° |
| M | -0.05 | 0.05 |
| X | 0 | 100 |

The objective function minimizes the average Euclidean distance between the predicted points generated by the parametric model and the ordered dataset.
